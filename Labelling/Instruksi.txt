Instruksi:
1. Download & Install Anaconda
2. Create and activate new environment in Anaconda
3. Tetap pada path direktori yang sama, Jalankan pada anaconda: conda install pyqt=5
4. Jalankan pada anaconda: conda install -c anaconda lxml
5. Masuk ke path direktori folder github pada link di atas.
6. Jalankan pada anaconda: pyrcc5 -o libs/resources.py resources.qrc
7. Jalankan pada anaconda: python labelImg.py [IMAGE_PATH (folder image)] [PRE-DEFINED CLASS FILE (Label.txt)]
8. Akan muncul windows app, pastikan pada view check auto save dan direktorinya sama pada direktori folder image
9. Mulai labelling, pastikan yang dikotakin seluruh fitur pada wajah kena. Setelah selesai semua label, lanjut ke picture berikutnya
10. Ulangi instruksi nomor 9 sampai 150 picture sudah di label.

Jika masih bingung, lihat Penjelasan.mp4

11. Setelah selesai upload foldernya dalam bentuk zip ke github branch refrences di folder Dataset Label yaa.
